# RAG 效果增强说明

## 本阶段已落地

1. 查询改写：基于银行办公术语同义词生成少量扩展查询，提升 BM25 召回。
2. 本地重排：综合 RRF 分数、标题路径命中、正文命中和切片质量分进行二次排序。
3. 检索追踪：响应中返回改写查询、各阶段耗时和命中数量，便于排查召回瓶颈。
4. 反馈闭环：新增 `/api/ai/retrieval/feedback`，沉淀用户评分、是否有帮助、意见和关联切片。
5. 评测样例：新增 `src/main/resources/rag/eval-samples.jsonl`，用于后续构建自动化 RAG 评测集。

## 后续增强计划

1. 将低命中问题和低分反馈自动汇总为知识补全清单。
2. 接入模型级 reranker，对 Top 50 融合结果做语义重排。
3. 将同义词词典外置到配置表或管理后台，并同步到 IK 自定义词典。
4. 建立离线评测任务，统计 Recall@K、MRR、引用命中率和低置信占比。
5. 在回答生成阶段增加强引用约束，禁止模型使用未检索到的制度依据。
